# Team Labels for Flutter Org PRs
Note:
- Some PRs are not scoped solely by labels. Pay attention to guidance below.
- PRs that are not scoped to a team should be listed as "Unowned", unless they are bot PRs like rolls, those should be classified as "Automated changes".
